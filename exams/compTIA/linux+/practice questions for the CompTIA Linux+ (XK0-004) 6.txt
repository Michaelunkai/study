You need to search for the occurrence of the word "error" in a file named "logfile.txt". Which command would you use?
a) grep "error" logfile.txt
b) search "error" logfile.txt
c) locate "error" logfile.txt
d) find "error" logfile.txt

a) grep "error" logfile.txt


You want to display all running processes sorted by their CPU usage,
with the most CPU-intensive processes listed first. Which command would you use?
a) ps -ef --sort=-%cpu
b) top -c
c) ps -aux --sort=-%cpu
d) pstree

a) ps -ef --sort=-%cpu

You need to find all files in the "/var/log" directory modified within the last 24 hours and list them along with their permissions.
Which command would you use?
a) find /var/log -mtime -1 -ls
b) ls -l /var/log/* -mtime -1
c) locate /var/log -mtime -1 -ls

in real-time as new lines are appended to it.
Which command would you use?

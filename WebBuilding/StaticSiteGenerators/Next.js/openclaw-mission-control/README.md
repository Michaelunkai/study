# 🎯 OpenClaw Mission Control

Real-time dashboard for monitoring and managing OpenClaw agent workflows.

## Features

- **📊 Live Agent Monitoring** - Track all active sessions and their current tasks
- **⚠️ Failure Tracking** - View and manage recurring failures with one-click fixes
- **🔨 Foundry Stats** - Monitor pattern learning, crystallization, and insights
- **⚡ Quick Actions** - One-click access to common workflows

## Tech Stack

- **Next.js 15** - React framework with App Router
- **TypeScript** - Type-safe development
- **Tailwind CSS** - Utility-first styling
- **React 19** - Latest React features

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the development server:
   ```bash
   npm run dev
   ```

3. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Development

- `npm run dev` - Start development server
- `npm run build` - Build for production
- `npm run start` - Start production server
- `npm run lint` - Run ESLint

# YouTube Filter v1.1 - Quick Reference Card

## ⚡ Keyboard Shortcuts
```
Ctrl+Shift+F  →  Toggle filter on/off
Ctrl+Shift+R  →  Refresh/refilter page
Ctrl+Shift+S  →  Show statistics
```

## 🎯 Right-Click Actions
```
Block this channel  →  Hide all videos from channel
Boost this channel  →  Highlight channel's videos
```

## 📊 Popup Tabs
```
Overview   →  Stats + Quick Actions
Settings   →  Basic Controls
Advanced   →  Filters + Blacklist
Help       →  Shortcuts + Export/Import
```

## 🔧 Quick Settings
```
Watch Threshold:  10% = Aggressive | 50% = Balanced | 90% = Conservative
Hide Shorts:      ON = No Shorts     | OFF = Show Shorts
Min Duration:     5min = No clips    | 0 = No limit
Max Duration:     30min = Short only | 0 = No limit
```

## 💾 Backup & Restore
```
Export:  Popup → Help → Export Data → Save JSON
Import:  Popup → Help → Import Data → Select JSON
```

## 📈 Analytics
```
File:  analytics.html (open in browser)
Shows: Filter stats, charts, time saved
```

## 🚨 Troubleshooting
```
Videos still showing?
  1. Lower watch threshold to 30%
  2. Click "Sync Watch History Now"
  3. Wait 10 seconds, refresh YouTube

Extension not working?
  1. Check chrome://extensions - should be ON
  2. Press F12 on YouTube, look for console message
  3. If errors, reload extension

Filter too aggressive?
  1. Raise threshold to 70%
  2. Or Ctrl+Shift+F to disable temporarily
```

## 🎯 Best Practices
```
✓ Start with 50% threshold
✓ Sync after watching many videos
✓ Use keyword blacklist sparingly (3-5 words max)
✓ Right-click boost your favorite channels
✓ Export settings monthly as backup
✓ Check analytics weekly to tune settings
```

## 💡 Pro Tips
```
→ Use Ctrl+Shift+F for quick toggle instead of clicking popup
→ Right-click boost channels you don't want to miss

→ Add "clickbait,shocking,must watch" to blacklist

```

## 🎨 Visual Indicators
```
Hidden videos:    Display: none (invisible)
Boosted videos:   Green border + shadow
Floating widget:  Bottom-right corner (🎯 + count)
```

## 📊 Stats Explained
```
Total Filtered:   Videos hidden all-time
Blocked:          Channels you've blocked
Boosted:          Channels you've boosted
Filter Rate:      % of videos filtered
Cache Hit Rate:   Performance metric (95%+ = good)
Time Saved:       Estimated hours saved (5min per video)
```

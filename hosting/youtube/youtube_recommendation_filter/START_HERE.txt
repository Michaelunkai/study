=== QUICK START (5 MINUTES) ===

STEP 1: INSTALL BROWSER EXTENSION
----------------------------------
1. Open Chrome or Edge
2. Type in address bar: chrome://extensions
3. Toggle ON "Developer mode" (top-right corner)
4. Click "Load unpacked" button
5. Navigate to this folder:
   F:\study\hosting\youtube\youtube_recommendation_filter
6. Click "Select Folder"
7. Extension now installed! ✓

STEP 2: ACTIVATE FILTERING
---------------------------
1. Click the extension icon in your browser toolbar
   (Look for red/black filter icon - pin it for easy access)
2. Click "🔄 Sync Watch History Now" button
3. Wait 5-10 seconds for sync to complete
4. Extension popup shows number of watched videos

STEP 3: VERIFY IT WORKS
------------------------
1. Open YouTube homepage: youtube.com
2. Press F5 to refresh the page
3. Videos you've watched should NO LONGER appear!
4. Check sidebar, search results - all filtered!

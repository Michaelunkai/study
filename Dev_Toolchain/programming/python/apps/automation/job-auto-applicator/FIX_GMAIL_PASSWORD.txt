================================================================================
                    GMAIL PASSWORD FIX - STEP BY STEP
================================================================================

PROBLEM: Your current Gmail app password is INVALID or EXPIRED.
         Gmail is rejecting it with authentication error 535.

SOLUTION: Generate a NEW Gmail app password (takes 2 minutes).

================================================================================
                           FOLLOW THESE STEPS:
================================================================================

STEP 1: ENABLE 2-STEP VERIFICATION (if not already enabled)
------------------------------------------------------------------------
1. Go to: https://myaccount.google.com/security
2. Click "2-Step Verification"
3. Follow the setup wizard
4. Verify your phone number

⚠️  NOTE: App passwords ONLY work if 2-Step Verification is enabled!


STEP 2: GENERATE NEW APP PASSWORD
------------------------------------------------------------------------
1. Go to: https://myaccount.google.com/apppasswords
2. Sign in if prompted
3. In the "App name" field, type: Job Applicator
4. Click "Create"
5. Google will show you a 16-character password like:
   
   abcd efgh ijkl mnop
   
6. COPY this password (click the copy icon or select and copy)


STEP 3: UPDATE CONFIG.PY WITH NEW PASSWORD
------------------------------------------------------------------------
Run this command (it will open config.py for you):

notepad "F:\study\Dev_Toolchain\programming\python\apps\automation\job-auto-applicator\config.py"

Find this line:
GMAIL_APP_PASSWORD = "rzzv mvbc bamy tpmr"  # Gmail App Password - "resume"

Replace it with your NEW password:
GMAIL_APP_PASSWORD = "abcd efgh ijkl mnop"  # Your new app password

Save and close the file (Ctrl+S, then close Notepad)


STEP 4: TEST THE NEW PASSWORD
------------------------------------------------------------------------
Run this command to test if it works:

python "F:\study\Dev_Toolchain\programming\python\apps\automation\job-auto-applicator\test_gmail_credentials.py"

You should see:
[SUCCESS] Gmail credentials are valid!


STEP 5: RUN THE JOB APPLICATOR
------------------------------------------------------------------------
Once the test passes, run this command:

python "F:\study\Dev_Toolchain\programming\python\apps\automation\job-auto-applicator\job_applicator_smtp.py"

It will send applications to all 7 companies with your resume!

================================================================================

A: You need to enable 2-Step Verification first

A: Make sure you copied the password correctly (no extra spaces)

Q: Gmail says "Username and Password not accepted"
A: Double-check the password in config.py matches what Google gave you

HOW THE APPLICATION WORKS:
---------------------------
- Enumerates all running processes
- For each process (except critical system ones):
  * Opens process handle with PROCESS_QUERY_INFORMATION and PROCESS_SET_QUOTA_INFORMATION
  * Calls EmptyWorkingSet() to release unused memory pages
  * Calls SetProcessWorkingSetSize() to minimize working set
- Clears system file cache using NtSetSystemInformation
- Shows total RAM freed in a message box
- Automatically closes after displaying results

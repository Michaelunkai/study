# Game Optimizer - Quick Start Guide

Game Optimizer is a system-tray application that automatically optimizes your Windows laptop for maximum gaming performance. It works by:

## Installation Steps

1. **Locate the Executable**
   - File: `GameOptimizer.exe`
   - Location: `F:\Downloads\GameMod\GameOptimizer.exe`

2. **Run as Administrator (REQUIRED)**
   - Right-click on `GameOptimizer.exe`
   - Select "Run as administrator"
   - Click "Yes" when prompted by UAC

3. **Look for the System Tray Icon**
   - After launching, find the icon in your system tray (bottom-right corner)
   - If you don't see it, click the up arrow (^) to show hidden icons

## How to Use

### To Enable Gaming Mode:
1. Right-click the system tray icon
2. Click "On"
3. Wait for confirmation message showing optimizations applied
4. Start your game and enjoy improved performance!

### To Disable Gaming Mode:
1. Right-click the system tray icon
2. Click "Off"
3. Wait for confirmation message showing settings restored
4. Your system returns to normal operation

### To Exit the Application:
1. Right-click the system tray icon
2. Click "Exit"
3. All optimizations are automatically reversed

## What Gets Optimized

### Background Processes (Set to Low Priority):
- Web browsers: Chrome, Firefox, Edge, Opera
- Chat apps: Discord, Teams, Skype, Slack
- Media: Spotify
- Cloud storage: OneDrive, Dropbox, Google Drive
- Windows system processes: Cortana, Search UI, Runtime Broker

- Power Plan ‚Üí High Performance

- Network Throttling ‚Üí Disabled
- GPU Priority ‚Üí Maximum for games

## Troubleshooting

**Problem**: Application won't start
**Solution**: Make sure you're running as Administrator (right-click ‚Üí Run as administrator)

**Problem**: No system tray icon visible
**Solution**: Check the hidden icons area (click ^ in system tray)

**Problem**: Optimizations don't seem to work
**Solution**: Verify administrator privileges and restart the application

**Problem**: System feels different after turning OFF
**Solution**: The application restores all original settings. Try restarting your computer if needed.

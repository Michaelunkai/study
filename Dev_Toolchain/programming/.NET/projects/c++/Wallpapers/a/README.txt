============================================
   WALLPAPER CHANGER - Automatic Wallpaper
============================================

DESCRIPTION:
This application automatically downloads and sets a high-resolution wallpaper
(3840x2160) from curated cyberpunk, gaming, sci-fi, and futuristic themes.

FEATURES:
- Instant wallpaper change with a single click
- No GUI or terminal windows
- Random selection from 24+ curated cyberpunk/sci-fi wallpapers
- Automatic fallback to high-quality random images if needed
- Highest resolution possible (4K - 3840x2160)

THEMES INCLUDED:
- Cyberpunk cities
- Gaming setups
- Futuristic landscapes
- Tech noir
- Dystopian futures
- Neon-lit cityscapes
- Sci-fi environments
- And more...

USAGE:
Simply double-click "wallpaper_changer.exe"

The application will:
1. Download a random high-resolution wallpaper
2. Automatically set it as your desktop background
3. Close immediately (no GUI)

You can run it as many times as you want - each time you'll get a different wallpaper!

FILES:
- wallpaper_changer.exe  : Main executable (just double-click it!)
- wallpaper.jpg          : Downloaded wallpaper (created automatically)
- build.bat              : Build script (if you want to recompile)
- wallpaper_changer.cpp  : Source code

NOTES:
- Requires internet connection to download wallpapers
- Wallpapers are cached in wallpaper.jpg
- Works on Windows 7/8/10/11

REBUILDING (Optional):
If you want to recompile from source:
1. Install MinGW or Visual Studio
2. Run: build.bat

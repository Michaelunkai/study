=================================================================
  WHAT HAPPENS AUTOMATICALLY
=================================================================

When you double-click INSTALL.cmd, it will:

1. ✅ Check if Windows Sandbox is enabled
   → If not: Enable it automatically (requires admin + restart)
   → If yes: Continue

2. ✅ Create isolated sandbox environment
   → Completely safe - nothing touches your main system

3. ✅ Launch Windows Sandbox with auto-run script
   → Opens in new window automatically

4. ✅ Inside the sandbox, it automatically:
   → Installs Node.js v24 LTS
   → Installs OpenClaw globally
   → Creates full config structure
   → Sets up system tray integration
   → Configures auto-start on boot
   → Installs Android Debug Bridge (ADB)
   → Creates helper scripts
   → Starts OpenClaw Gateway
   → Runs 10 verification checks

5. ✅ Shows you the entire installation process in real-time

6. ✅ When you close the sandbox window:
   → Everything is automatically discarded
   → No cleanup needed
   → Zero impact on main system
